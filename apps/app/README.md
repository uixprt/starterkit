# @a/app

React application built with Vite and TypeScript that demonstrates the use of the module-header and design-system packages.

## Features

- React 18
- TypeScript
- Vite for fast development
- Material-UI integration
- Theme switcher
- Uses @a/module-header for consistent header
- Integrates @uixprt/elements web components

## Development

```bash
npm run dev
```

Opens the app at http://localhost:3000

## Build

```bash
npm run build
```

Builds the production-ready app to the `dist` folder.

## Preview

```bash
npm run preview
```

Preview the production build locally.
